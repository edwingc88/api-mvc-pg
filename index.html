API Laboratorio Clinico "San Rafael"

API Lab ☕︎

  • GET : Consulta todos los Laboratorios 🗹

    EndPoint : https://lab-san-rafael-api.onrender.com/labs

  • POST : Crear Un Laboratorio 🗹

    EndPoint : https://lab-san-rafael-api.onrender.com/labs

    Headers : multipart/form-data;

      {
  "name":"Centro Médico Ambulatorio San Rafael2",
  "rif":"J-12345678",
  "slogan":"Los resultados en los que puede confiar",
  "description":" Un laboratorio clínico es una especialidad médica que permite contribuir al estudio, prevención, diagnóstico y tratamiento de las enfermedades humanas.",
  "address":"65JG+H7V, Ciudad Guayana 8050, Bolívar. Urb Gran Sabana",
  "phone":"0286-9530505",
  "email":"lab@gmail.com",
  "logo":"localhost/url/imagen.jpg",
  "objetive":"objetivos de lab",
  "mission":"misiones de lab",
  "vision":"vision de lab"
  }

  • PATH : Actualizar Laboratorio por ID 🗷

    https://lab-san-rafael-api.onrender.com/labs/:id

  • PATH : Actualizar Imagen por ID 🗷

    https://lab-san-rafael-api.onrender.com/labs/updateimg/:id

  • DELETE : Eliminar Lab por ID 🗹

    EndPoint : https://lab-san-rafael-api.onrender.com/labs/:id

  • DELETE : Eliminar Imagen por ID 🗹

    EndPoint : https://lab-san-rafael-api.onrender.com/labs/deleteimg/:id
